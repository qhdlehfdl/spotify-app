# 🎵 Statify 📊

---
## 프로젝트 소개
- Statify는 [스포티파이 차트](https://charts.spotify.com/charts/overview/global) 데이터를 이용한 다양한 데이터 시각화 웹사이트입니다.
- 다양한 나라의 장르, 인기 아티스트, 차트 등을 볼 수 있습니다.
